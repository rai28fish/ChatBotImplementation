ChatBot Widget Example

This page demonstrates the embedded AI chatbot widget. Look for the chat button in the bottom-right corner.

One-Line Integration

To add this chatbot to any website, paste this single line before </body>:

<script
  src="http://localhost:3000/widget/chatbot-widget.js"
  data-chatbot-id="10b4b121-57c3-4f17-a2f1-73c21e8bf321"
  async
></script>

How to Get Your Chatbot ID

Create a chatbot via the API:

curl -X POST http://localhost:3000/create-chatbot \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Company Bot",
    "baseUrl": "https://yourcompany.com",
    "crawlOptions": { "maxPages": 50, "maxDepth": 2 },
    "config": {
      "primaryColor": "#0066cc",
      "welcomeMessage": "Hi! How can I help you today?"
    }
  }'

The response includes a chatbotId — use that in the embed script above.

Check Indexing Status

curl http://localhost:3000/chatbot/YOUR_CHATBOT_ID/status

Wait until "status": "ready" before embedding the widget.
